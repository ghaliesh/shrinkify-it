## Installation :

```
npm install
```

```
cd client && npm install
```

or

```
yarn install
```

```
cd client && yarn install
```

## Running the app :

```
npm -g install nodemon
```

```
npm run dev
```

or

```
yarn global add nodemon
```

```
yarn run dev
```

- [Angular](https://angular.io) - Frontend
